###################
SiMangat Resto & Chill
###################

**SiMangat Resto & Chill** adalah aplikasi sistem informasi pemesanan restoran berbasis web yang dirancang untuk membantu proses operasional restoran secara terstruktur dan efisien. Aplikasi ini mendukung pengelolaan menu makanan dan minuman, pemesanan, reservasi meja, serta konfirmasi pembayaran oleh pelanggan.

Sistem ini dibangun menggunakan framework **CodeIgniter 3** dengan bahasa pemrograman **PHP**, serta didukung oleh **MySQL** sebagai basis data. Antarmuka dirancang menggunakan **HTML, CSS, dan JavaScript** untuk memberikan tampilan yang sederhana, responsif, dan mudah digunakan.

Aplikasi SiMangat Resto & Chill ditujukan sebagai media pendukung pelayanan restoran agar proses pemesanan dan pengelolaan data dapat dilakukan secara lebih cepat, akurat, dan terintegrasi. Proyek ini dikembangkan sebagai bagian dari tugas akademik dan dapat dijadikan referensi pembelajaran dalam pengembangan sistem informasi berbasis web.
